# Netflix-Analytics-Project
Netflix data analysis using Python and Power BI


Tools Used:
- Python (Pandas, NumPy)
- Power BI

Workflow:
1. Data cleaning
2. Data analysis
3. Visualization in Power BI
4. Dashboard creation
5. Insight generation

Key Insights:
1.Movies dominate Netflix with 69.61% of total content
2.TV Shows are the least in number with only 2676 titles
3.Most content is produced from United States
4.Least content is produced from Canada among top countries
5.Content is increasing over time, peaking around 2019-2020
6.Most content is released in the 2015-2020 year range
7.Movies are more in number than TV Shows
8.Rating TV-MA is the most common across all content
9.India is the 2nd largest content producer after USA
10.Content before 2000 is very rare on Netflix
11.TV-14 is the 2nd most common rating after TV-MA
12.Netflix added the most content around 2019 before declining
13.Over 69% of Netflix library consists of Movies, not TV Shows
